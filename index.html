Binary — Quick Overview

Binary is a base-2 number system that uses only two digits, 0 and 1, to represent all information.
How binary works
  • Base-2 system: Unlike decimal (base-10), binary has two digits: 0 and 1.
  • On/Off states: Electronic circuits use 0 and 1 to represent off/on.
  • Bits & bytes: A single binary digit is a bit. Eight bits = 1 byte.
Example: 2 in binary is 10. 3 is 11.
